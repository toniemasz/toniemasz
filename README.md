**Technologie:** Django, Python, baza danych, Git/GitHub

Aplikacja webowa używana w gospodarstwie przez jednego użytkownika do zarządzania danymi i zadaniami. Projekt jest aktywnie rozwijany oraz aktualizowany.

Zakres projektu obejmuje:

* pracę nad formularzami, widokami i modułami zarządzania,
* obsługę danych w bazie danych,
* poprawki UX/UI,
* rozwój funkcji pod realne potrzeby użytkownika,
* nacisk na czytelność i użyteczność systemu.

### WCPredictorGame

**Technologie:** Python, API, baza danych, Git/GitHub

Aplikacja/gra ze znajomymi do przewidywania wyników meczów. System pobiera mecze z API i nalicza punkty za trafne typy.

Zakres projektu obejmuje:

* pobieranie danych o meczach z zewnętrznego API,
* logikę typowania i punktacji,
* przechowywanie typów użytkowników,
* obsługę kilku aktywnych użytkowników,
* pracę z bazą danych.

### Przewidywanie wyników meczów piłki nożnej

**Technologie:** Python, Pandas, scikit-learn, Machine Learning, analiza danych

Projekt analityczno-predykcyjny związany z pracą licencjacką. Celem projektu było przygotowanie danych sportowych oraz budowa modeli klasyfikacyjnych służących do oceny prawdopodobieństwa wyników spotkań piłkarskich.

Zakres projektu obejmuje:

* samodzielne pozyskanie i przygotowanie danych sportowych,
* czyszczenie danych,
* tworzenie zmiennych i cech modelowych,
* analizę danych historycznych,
* porównywanie podejść modelowych,
* wykorzystanie metod predykcyjnych do przewidywania wyników meczów.
